# Vizor Wallet v0.0.32

This release adds support for importing secret phrases that have activity
across multiple ZIP32 account indices.

## What's New

- Wallet import can now find additional ZIP32 software accounts with
  transparent activity and offer to import them alongside the primary account.
- Added an additional-account selection step during import, with transparent
  balance previews and BIP44 account paths to make each discovered account
  easier to identify.
- Re-importing a secret phrase that is already in the wallet can now add missing
  higher-index accounts without duplicating the existing primary account.

## Notes

- Additional-account discovery is based on transparent account activity, so
  accounts without detected transparent history may not appear automatically.
