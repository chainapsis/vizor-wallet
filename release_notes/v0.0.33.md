# Vizor Wallet v0.0.33

This release introduces a full redesign of Vizor Wallet. The app is now cleaner, easier to navigate, and more consistent across the main wallet experience.

## What's New

- Redesigned the app experience across onboarding, home, accounts, contacts, activity, receive, send, swap, Keystone, and settings.
- Refreshed Vizor's visual system with updated typography, colors, controls, modals, and desktop layout behavior.
- Added ZEC price context on the home screen and during sends.

## Improvements

- Made wallet setup, navigation, transaction review, and status screens easier to follow.
- Improved readability and consistency across activity history, contacts, QR/signing flows, and balance displays.
