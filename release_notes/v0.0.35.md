# Vizor Wallet v0.0.35

This release improves transparent address privacy, sync recovery, and
transaction status clarity on desktop.

## What's New

- Transparent receive addresses now rotate automatically after receiving ZEC,
  while previous addresses remain part of the wallet.

## Improvements

- Receive screens now explain transparent address rotation and refresh the
  displayed transparent address after sync updates.
- Transaction status screens now show shorter transaction IDs for easier
  reading.
- The Vote screen now labels voting as beta.

## Fixes

- Fixed wallets getting stuck at 0% syncing after deleting an imported account
  with an old birthday. Affected wallets can recover by updating and syncing
  again.
- Fixed transparent-to-transparent received transactions being labeled as
  shielded in the transaction status view.
