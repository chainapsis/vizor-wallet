# Vizor Wallet v0.0.36

This release improves account management, swap guidance, and Keystone setup
across the desktop app.

## Improvements

- Multi-account wallets can now remove the original software account; removing
  the final account still resets Vizor.
- Swap quote checks are stricter about amounts, addresses, slippage, and route
  details before a quote reaches review.
- Swap errors now give clearer guidance for low amounts, unsupported assets,
  unavailable liquidity, and insufficient spendable ZEC for fees.
- Keystone setup now has clearer firmware and connection steps, including a
  direct firmware link.

## Fixes

- Fixed some valid 1Click swap quotes being rejected before review when the
  provider returns normalized route information.
