# Vizor Wallet v0.0.24

This release makes first-time setup more flexible, improves account naming, and
fixes several send and activity display issues.

## What's New

- Added endpoint settings to the first welcome screen, so you can choose a
  custom lightwalletd endpoint before creating or importing a wallet.

## Improvements

- Account names can now be as short as 1 character and are validated by
  user-perceived characters, so emoji and other multi-codepoint characters count
  correctly.

## Fixes

- Fixed Activity details for sent transactions so recipient addresses are shown
  more accurately, including transparent sends and explicit self-transfers.
- Fixed the send status screen so long shielded recipient addresses fit in the
  receipt layout more reliably.
- Fixed text field shells so clicking the field area focuses the input as
  expected.
