# Vizor Wallet v0.0.31

This release adds TEX address sending, improves transaction history fees for
TEX and ZIP-320 send flows, and handles unsupported Keystone TEX sends more
clearly.

## What's New

- Added support for sending ZEC to TEX addresses, including address validation,
  Send review labeling, and transaction history support.

## Improvements

- Transaction history now reports fees more accurately for TEX and ZIP-320 send
  flows.

## Fixes

- Fixed Keystone TEX swap deposit handling by showing a clear unsupported-flow
  message instead of starting an unsupported send.

## Notes

- TEX recipients behave like transparent recipients, so messages are not
  available for TEX sends.
- Keystone does not support TEX sends yet.
